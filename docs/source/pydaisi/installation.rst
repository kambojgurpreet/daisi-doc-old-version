Installing pyDaisi
============================

pyDaisi currently supports Linux, MacOS and Windows.

.. note::
    
    pyDaisi requires Python 3.7 or higher.

Official Releases
-----------------

PIP
^^^^^^^^^^^^^^^^

If you use `pip`, you can install the latest official version of pyDaisi with : 

.. code-block:: shell

  pip install --upgrade pydaisi
